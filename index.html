Automated waste to compost converter

The Automated Waste-to-Compost Converter is an ESP32-based IoT system designed to convert organic waste into compost efficiently. It uses sensors like a gas sensor (to detect harmful gases like methane and COâ‚‚), moisture sensor (to maintain optimal moisture levels), temperature sensor (to monitor heat during decomposition), pH sensor (to track acidity levels), and an ultrasonic sensor (to detect waste levels inside the bin). A servo motor automatically opens and closes the lid for waste input. The ESP32 processes data and sends it to a cloud dashboard such as Blynk or ThingSpeak for real-time monitoring. If Wi-Fi is unavailable, a buffer memory temporarily stores data to prevent loss. Alerts are generated for high gas levels, improper pH, or when the bin is full. This system promotes eco-friendly waste management, making it ideal for households, farms, and smart cities by producing organic compost automatically.
